# Plano - padroes mobile de UX

## Objetivo

Implementar ajustes de consistencia de interface em uma branch local, sem push,
mantendo o app mobile-first e preservando a paleta visual atual.

## Escopo

- Padronizar avisos e feedbacks visuais simples.
- Corrigir affordances que parecem navegacao mas nao sao.
- Dar uma saida clara no preview de importacao.
- Reduzir risco de finalizar treino incompleto por engano.
- Melhorar o sheet de registro de resultado sem adicionar dependencia nova.

## Done when

- Build e lint passam.
- Fluxo mobile principal continua funcionando: importar plano, iniciar treino,
  registrar um exercicio, tentar finalizar e navegar por historico/ajustes.
- Commit local criado na branch `ux/padroes-mobile`.
- Nenhum push executado.
